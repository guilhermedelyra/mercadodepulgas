# Sobre
Site de venda de móveis e mobílias para casa.

## Pre Requisitos
* Ruby 2.3.1
* Rails 4.2.6
* PostgreSql 9.5
* Spree 3.1.0
* Spree Static Content 3.1.0
* Spree i18n 3.1.0

## Configurações

* É necessário configurar os acessos ao banco de dados PostgreSql no arquivo config/database.yml

## Rodando o server

```
rake db:create
```
```
rake db:migrate
```
```
rake db:seed
```
```
rails s
```
